Gauges

A gauge is a visualization that shows a linear quantity as a percentage of a total. Gauges are often used to visualize occupancy or progress.

To add a gauge to a report, select the report cell where you want the gauge to appear. Then click the  Gauge Wizard icon. Proceed through the Appearance and Data pages, then click Finish when you are done.

To edit a gauge, double-click it, or select it and click the  Gauge Wizard icon to reopen the wizard.

Data

Gauges are a linear scale that is usually divided into ranges, each representing a category of meaning. In the Data page, set the data point and data ranges of the scale. Ranges can be fixed or based on dynamic data values.

Value and Range

Select a cell from the list to set the gauge data value.

To set the range, choose one of the following range types:

  • To use a fixed range, select the Static Value check box. Then enter the min and max values for the range, as whole numbers.

  • To use a dynamic range, select the Cell Value check box. Then select the cells to supply the min and max values for the range.

  • To use a partially dynamic range, select the Cell Value check box. Select the cell to supply the dynamic endpoint value. Then for the fixed endpoint, select a report cell containing a fixed numeric value, as written with a preceding equal sign (=). For example: =100.

Color Ranges

Colors are used to classify and distinguish ranges of data. Ranges are set by percentage, fixed values, or dynamic values. A Bulb gauge must have two or more ranges in order to be meaningful. Thermometer gauges have only one color.

To add color ranges to a gauge:

  1. Click  Add for each range to add.

  2. Choose how to set the ranges:

     Tip
    Each value is the upper endpoint for that range, which starts from the end of the previous one. Fixed values must be whole numbers in ascending order. Values cannot exceed the max value of the gauge. The last range's upper endpoint is the end of the gauge. Ranges will not overlap.

    • To use percentage ranges, select the Percentage check box. Then enter a value for each range.

      Percentage ranges are fixed proportions on the gauge, irrespective of the total range or data value.

    • To use fixed ranges, select the Static Value check box. Then enter a value for each range.

    • To use dynamic ranges, select the Cell Value check box. Then, for each range, select a cell to supply the value.

      Dynamic ranges are ordered by value, not by the specified order.

    • To use partially dynamic ranges, select the Cell Value check box. For each dynamic range, select a cell to supply the value. For each fixed range, select a report cell containing a fixed numeric value, as written with a preceding equal sign (=). For example: =100.

  3. For each range, enter a color code or use the color picker to select a color.

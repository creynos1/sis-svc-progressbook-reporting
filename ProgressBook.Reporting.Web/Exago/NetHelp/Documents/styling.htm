Styling

ExpressViews can be styled in a variety of ways. Click the Report Formatting and Style  icon to see your available options.

Using a premade theme

If there are any available, you can select a premade theme to use for the ExpressView, or to use as a baseline for further customizations. On the Style page, click the Theme tab. Then select a theme from the Theme Selector list:

Legacy

This is the default theme.

Custom

This option indicates that you are not currently using a premade theme.

Selecting a premade theme overrides any custom styling you have. Be sure to save the ExpressView first, so that you can retrieve your styling if you accidentally override it.

Styling data cells

Data columns and group columns can be styled independently, using the Style tab on the Selected Cell page. You can style the data cells, as well as group headers and footers.

To style a data cell:

  1. With the Selected Cell page open, click the section, header, or footer you want to style. It will highlight in blue.

     

    Selecting an element to format

  2. Click the Style tab.

  3. Choose a look for the cell text and background.

You can also use the Selected Cell page to edit the header names of the data columns. To do so:

  1. With the Selected Cell page open, click the header which you want to edit. It will highlight in blue.

  2. Click the Edit icon on the header.

  3. Type a name for the header.

     

    Changing header text

  4. Press Enter.

Changing data row colors

The background colors for the data rows can be customized, and you can set the pattern by which the colors alternate. This affects every data section in the ExpressView.

To set the background colors:

  1. On the Style page, click the Row Shading tab. By default there are two colors which alternate every other row.

  2. Choose the number of colors that you want to alternate between:

    • To make all rows the same color, click to delete all the colors except one.

    • To make rows alternate between more than two colors, click to add more colors.

  3. Enter a color code or use the color picker to set each color.

Changing group colors

The headers and footers for group columns are prefixed by a different color depending on the level of grouping. These colors can be customized, and you can set the pattern by which the colors alternate. This affects every group header and footer in the ExpressView.

To set the group level colors:

  1. On the Style page, click the Group Colors tab. By default, nested groups alternate between four colors.

  2. Choose the number of colors that you want to alternate between:

    • To make all group levels the same color, click to delete all the colors except one.

    • To make group levels alternate between more than four colors, click to add more colors.

  3. Enter a color code or use the color picker to set each color.

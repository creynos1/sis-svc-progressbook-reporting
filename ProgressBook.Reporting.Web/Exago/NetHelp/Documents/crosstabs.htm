Crosstabs

A crosstab, or cross tabulation, is a widget for grouping and summarizing data fields that automatically expands vertically and horizontally depending on the number of data groupings. Crosstabs are also known as two-way tables, contingency tables, and pivot tables.

Using a crosstab on a report will prevent you from using any of the cells below or to its right. Therefore, it is recommended to use crosstabs as the main data display for the reports they are on.

To add a crosstab to a report:

  1. Select a report footer cell, then click the Crosstab Wizard  icon.

  2. To add rows, drag data fields to the Row Header Source pane. To add columns, drag data fields to the Column Header Source pane.

    Adding more than one row or column creates data groups from the selected fields. Drag the rows up or down to select how they are grouped.

  3. Select which field to collect data by dragging the field to the Tabulation Data Source pane.

  4. Optional: To use a formula instead of a data field, click the formula icon and enter a formula.

  5. Optional: Click the edit icon to select options for a data field:

    • Label: Edit the title of the field.

    • Method: Select how to sort the data. Select a Direction.

    • Placement: Select whether to show a Total and Subtotal, and where to display them.
